# LoanFlow — Microservices Loan Origination System

  A learning project demonstrating microservices architecture, event-driven design, and cloud-native deployment patterns for banking systems.

  **Status: In active development**

  ## Planned architecture
  - 6 Spring Boot services (customer, loan-application, credit-check, notification, api-gateway, discovery-server)
  - PostgreSQL for persistence, Kafka for async event flow
  - Containerized with Docker, orchestrated with docker-compose
  - JWT auth via API Gateway
  - Deployment target: AWS

  Detailed architecture and decisions: see /docs (coming soon).
